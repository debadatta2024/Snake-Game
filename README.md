# 🐍 Snake Game

A simple Snake Game built using HTML, CSS, and JavaScript.

## 🎮 Play the Game
👉 https://debadatta2024.github.io/Snake-Game/

## 🚀 Features
- Keyboard controls (Arrow keys)
- Score & High Score (localStorage)
- Timer
- Game Over & Restart
- Smooth snake movement

## 🕹 Controls
- ⬅️ Left Arrow → Move Left
- ➡️ Right Arrow → Move Right
- ⬆️ Up Arrow → Move Up
- ⬇️ Down Arrow → Move Down

## 📁 Tech Stack
- HTML
- CSS
- JavaScript
